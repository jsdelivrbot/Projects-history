Frood Web Application
=====================

### Detailed Info

### Usage

```
npm install
npm run dev
open http://localhost:3000
```

Now edit any component in the app.
Your changes will appear without reloading the browser.

### Linting

This boilerplate project includes React-friendly ESLint configuration (AirBNB).

```
npm run lint
```

### Using `0.0.0.0` as Host

### Dependencies

* React
